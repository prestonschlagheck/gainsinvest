G.AI.NS API Configuration
============================

Please add your API keys below:

1. OpenAI API Key:
   OPENAI_API_KEY=your_openai_api_key_here

2. Grok API Key:
   GROK_API_KEY=your_grok_api_key_here

3. Market Data API (Choose one):
   
   Option A - Alpha Vantage (Free tier: 5 API requests per minute, 500 per day)
   ALPHA_VANTAGE_API_KEY=your_alpha_vantage_api_key_here
   Get free key at: https://www.alphavantage.co/support/#api-key
   
   Option B - Finnhub (Free tier: 60 API calls/minute)
   FINNHUB_API_KEY=your_finnhub_api_key_here
   Get free key at: https://finnhub.io/register

Instructions:
1. Copy this file to .env.local in the root directory
2. Replace the placeholder values with your actual API keys
3. Never commit .env.local to version control

Example .env.local format:
OPENAI_API_KEY=sk-...
GROK_API_KEY=gsk_...
ALPHA_VANTAGE_API_KEY=ABC123...
NEXTAUTH_SECRET=your-secret-key-here
